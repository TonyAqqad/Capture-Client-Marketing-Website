## ANIMATION TIMELINE

```
PAGE LOAD:
0ms   │ Background mesh fades in
100ms │ Floating orbs start pulsing
200ms │ Eyebrow "Get Started" fades in
300ms │ Headline animates in
400ms │ Form card appears
500ms │ Contact cards appear
600ms │ Timeline section visible
700ms │ Service cards appear

CONTINUOUS:
∞     │ Form border shifts colors (8s loop)
∞     │ Phone icon pings (1.5s loop)
∞     │ Location icon pulses (2s loop)
∞     │ Orbs float (8s loop each)
∞     │ "Available Now" dot pings (1s loop)

ON HOVER:
0ms   │ Transition starts
300ms │ Glow reaches full opacity
500ms │ Transition complete
```

## SPACING & SIZING

### Typography Scale
```
H1 (Headline):     5xl → 6xl → 7xl (mobile → tablet → desktop)
H2 (Sections):     3xl → 4xl → 5xl
H3 (Cards):        xl → 2xl
Phone Number:      2xl → 3xl
Body Text:         base → lg → xl
Small Text:        xs → sm
```

### Spacing
```
Section Padding:   py-20 (80px top/bottom)
Container:         px-6 → px-8 (mobile → desktop)
Card Padding:      p-8 → p-10
Gap Between:       gap-6 → gap-12 (cards → columns)
```

### Border Radius
```
Cards:            rounded-2xl (16px)
Buttons:          rounded-xl (12px)
Badges:           rounded-full (pill)
```
