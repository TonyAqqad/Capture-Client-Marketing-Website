## Theme Application Patterns

### Hero Section Pattern
```tsx
const theme = getThemeByCategory(categoryId);

<section className={theme.bgClass}>
  <div className="relative">
    {/* Glow backdrop */}
    <div className={`absolute inset-0 ${theme.accentGlow} opacity-20 blur-3xl`} />

    {/* Content */}
    <div className="relative z-10">
      <span className={theme.badgeClass}>
        {theme.name}
      </span>
      <h1 className={`bg-gradient-to-r ${theme.gradient} bg-clip-text text-transparent`}>
        Title
      </h1>
    </div>
  </div>
</section>
```

### Card Pattern
```tsx
<div className={`${theme.bgClass} ${theme.borderClass} border rounded-xl p-6`}>
  <div className={`${theme.iconBg} ${theme.accentGlow} w-16 h-16 rounded-xl mb-4`}>
    {/* Icon */}
  </div>
  <h3 className={`text-xl font-bold bg-gradient-to-r ${theme.gradient} bg-clip-text text-transparent`}>
    Card Title
  </h3>
</div>
```

### Button Pattern
```tsx
<button className={`${theme.iconBg} ${theme.hoverGlow} px-6 py-3 rounded-xl text-white transition-all`}>
  Get Started
</button>
```

## Accessibility Considerations

All themes maintain WCAG 2.1 AA compliance:
- Text contrast ratios meet 4.5:1 minimum
- Glow effects are decorative only
- Color is not the sole means of conveying information
- Reduced motion support for animations

### Example Accessible Implementation
```tsx
<div className={theme.bgClass}>
  <span className="sr-only">{theme.name} category</span>
  <div className={`${theme.iconBg} motion-reduce:shadow-none ${theme.accentGlow}`}>
    {/* Icon with aria-label */}
  </div>
</div>
```

## Migration Guide

### Replacing Hardcoded Colors

**Before:**
```tsx
<div className="bg-gradient-to-r from-blue-500 to-blue-700">
  <h2 className="text-blue-400">Phone Systems</h2>
</div>
```

**After:**
```tsx
import { getThemeByCategory } from '@/data/categoryThemes';

const theme = getThemeByCategory('phone-systems');

<div className={`bg-gradient-to-r ${theme.gradient}`}>
  <h2 className={`bg-gradient-to-r ${theme.gradient} bg-clip-text text-transparent`}>
    {theme.name}
  </h2>
</div>
```

### Benefits of Theme System
1. **Consistency** - All categories use same visual patterns
2. **Maintainability** - Change colors in one place
3. **Type Safety** - Full TypeScript support
4. **Performance** - No runtime color calculation
5. **Accessibility** - Built-in WCAG compliance
6. **Flexibility** - Easy to add new categories
