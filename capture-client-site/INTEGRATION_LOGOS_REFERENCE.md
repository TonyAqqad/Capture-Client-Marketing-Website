## Integration Key Naming Rules

### ‚úÖ Correct Format
- Lowercase only: `'hubspot'` not `'HubSpot'`
- Hyphenated: `'google-calendar'` not `'google_calendar'` or `'googlecalendar'`
- No spaces: `'microsoft-teams'` not `'microsoft teams'`
- Match key exactly: `'housecall-pro'` not `'housecallpro'`

### ‚ùå Common Mistakes
```tsx
// Wrong
<IntegrationLogo integration="HubSpot" />
<IntegrationLogo integration="google calendar" />
<IntegrationLogo integration="ServiceTitan" />

// Correct
<IntegrationLogo integration="hubspot" />
<IntegrationLogo integration="google-calendar" />
<IntegrationLogo integration="servicetitan" />
```

## Category Helper Functions

Use these functions to get integrations by category:

```typescript
import { getIntegrationsByCategory } from '@/data/integration-logos';

// Get all CRM logos
const crmLogos = getIntegrationsByCategory('crm');

// Get all communication logos
const commLogos = getIntegrationsByCategory('communication');

// Get all field service logos
const fieldServiceLogos = getIntegrationsByCategory('fieldService');

// Get all scheduling logos
const schedulingLogos = getIntegrationsByCategory('scheduling');

// Get all payment logos
const paymentLogos = getIntegrationsByCategory('payments');

// Get all marketing logos
const marketingLogos = getIntegrationsByCategory('marketing');
```

Available categories: `'automation' | 'crm' | 'scheduling' | 'communication' | 'fieldService' | 'payments' | 'marketing' | 'legal' | 'analytics'`

## Logo URL Patterns

All logos follow this pattern:
```
https://logo.clearbit.com/{domain}
```

Examples:
- Zapier: `https://logo.clearbit.com/zapier.com`
- HubSpot: `https://logo.clearbit.com/hubspot.com`
- Slack: `https://logo.clearbit.com/slack.com`

**Note**: Clearbit automatically returns the correct logo for the domain. No API key required.

## Alternative Logo URLs

Some integrations have official alternatives (higher quality, SVG):

### Slack
```typescript
const slack = getIntegrationLogo('slack');

// Primary (Clearbit)
slack.logoUrl // 'https://logo.clearbit.com/slack.com'

// Alternatives (Official)
slack.alternativeUrls.svg   // SVG version
slack.alternativeUrls.white // White version (dark backgrounds)
slack.alternativeUrls.png   // Official PNG
```

To use alternatives:
```tsx
import { getIntegrationLogo } from '@/data/integration-logos';

const slack = getIntegrationLogo('slack');
const logoUrl = slack?.alternativeUrls?.svg || slack?.logoUrl;

<img src={logoUrl} alt="Slack logo" />
```

## Adding New Integrations

1. Open `src/data/integration-logos.ts`
2. Add new entry:
```typescript
'new-integration': {
  name: 'New Integration',
  logoUrl: 'https://logo.clearbit.com/newintegration.com',
  source: 'Clearbit Logo API',
  brandGuidelines: 'https://newintegration.com/brand', // Optional
},
```
3. Add to category (optional):
```typescript
export const integrationCategories = {
  // ...
  yourCategory: ['new-integration'],
};
```
4. Use immediately:
```tsx
<IntegrationLogo integration="new-integration" size="md" />
```
