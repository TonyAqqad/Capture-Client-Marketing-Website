## Design System Details

### Typography Scale
```
Desktop Headline: text-7xl (4.5rem / 72px)
Tablet Headline: text-6xl (3.75rem / 60px)
Mobile Headline: text-5xl (3rem / 48px)

Desktop Subheadline: text-2xl (1.5rem / 24px)
Mobile Subheadline: text-xl (1.25rem / 20px)
```

### Spacing
```
Hero Min Height: 90vh
Content Padding X: px-6 (mobile), px-16 (desktop)
Content Max Width: 5xl (64rem / 1024px)
Icon Size (Main): w-20 h-20 (5rem / 80px)
Icon Size (Floating): w-16 h-16 (4rem / 64px)
```

### Border Radius
```
Main Icon Container: rounded-2xl (1rem / 16px)
Floating Icons: rounded-2xl
CTA Buttons: rounded-full
Stats Cards: rounded-2xl
```

### Backdrop Effects
```
Glass-morphism: backdrop-blur-sm, backdrop-blur-md
Border Transparency: border-white/10, border-white/20, border-white/30
Background Transparency: bg-white/5, bg-white/10, bg-white/20
```

### Shadow Effects
```
CTA Button: shadow-2xl
Stats Cards: border glow on hover
Floating Icons: subtle box-shadow in theme colors
```

## Animation Timing

### Entrance Animations
```
Initial Delay: 0.2s
Item Stagger: 0.1s per item
Word Stagger: 0.1s per word (starting at 0.3s)
Stats Stagger: 0.15s per stat (starting at 0.8s)
```

### Continuous Animations
```
Background Gradient: 10s loop
Floating Icons: 6s float cycle
Stats Pulse: 3s pulse cycle
Scroll Indicator: 1.5s bounce cycle
```

### Interaction Animations
```
CTA Hover: 0.3s shimmer
CTA Scale: Spring physics (stiffness: 400, damping: 17)
CTA Tap: 0.98 scale
```

## Accessibility

### Color Contrast
All text on gradient backgrounds meets WCAG AA standards:
- White text on dark gradients: 7:1+ contrast ratio
- Large text (24px+): Meets AAA standards
- CTA buttons: High contrast with background

### Focus States
- All interactive elements have focus indicators
- Keyboard navigation supported
- Screen reader friendly

### Motion
- Respects `prefers-reduced-motion`
- Critical animations can be disabled
- No motion-required functionality

---

## Responsive Breakpoints

### Mobile (< 768px)
- Single column layout
- Floating icons hidden
- Reduced animation complexity
- Larger touch targets (min 44px)
- Stats stack vertically

### Tablet (768px - 1024px)
- 2-column stats grid
- Medium floating icons
- Scaled typography
- Optimized spacing

### Desktop (> 1024px)
- Full floating icons visible
- 3-column stats grid
- Maximum visual impact
- All animations active

## Usage Examples

### Implementing a New Service Theme

```typescript
// Add to serviceThemes object in ServiceHero.tsx
"new-service": {
  gradient: "from-[color1] via-[color2] to-[color3]",
  accentColor: "bg-[color]",
  iconColor: "text-[color]",
  icon: YourIcon,
  floatingIcons: [
    { Icon: Icon1, delay: 0, position: "top-20 right-20" },
    { Icon: Icon2, delay: 0.2, position: "top-40 left-20" },
    { Icon: Icon3, delay: 0.4, position: "bottom-32 right-32" },
  ],
}
```
