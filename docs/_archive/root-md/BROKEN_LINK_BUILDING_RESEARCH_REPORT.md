This report identifies 25+ high-value broken link building opportunities across marketing agency directories, AI tool lists, lead generation resource pages, and small business directories. ## Part 3: Broken Link Finding Methods & Tools

### A. Manual Research Methods

#### Method 1: Search Operators for Resource Pages
```
"voice ai" + "recommended tools" + "resources"
"lead generation" + "best agencies" + inurl:resources
"marketing agency" + intitle:"resource page" + Tennessee
"small business tools" + "recommended" + 2022
```

#### Method 2: Find Outdated Lists
```
"best marketing tools" + 2022
"top lead generation software" + 2023
"ai tools directory" + "last updated 2022"
```

#### Method 3: Competitor Backlink Analysis
```
Use Ahrefs/SEMrush to find backlinks to competitors
Filter by "404 not found"
Identify resource pages linking to broken content
Pitch Capture Client as replacement
```

### B. Automated Tools for Broken Link Building

#### 1. **Ahrefs Site Explorer**
- **Purpose:** Find competitor broken backlinks
- **Process:**
  1. Enter competitor URL (e.g., MyAIFrontDesk.com)
  2. Go to "Backlinks" > Filter by "404 not found"
  3. Export list of referring domains
  4. Create similar content, pitch as replacement

#### 2. **Ahrefs Broken Link Checker**
- **Purpose:** Check any page for broken outbound links
- **Process:**
  1. Enter resource page URL
  2. Scan for broken links
  3. Identify relevant broken links
  4. Create replacement content

#### 3. **Check My Links (Chrome Extension)**
- **Purpose:** Visual broken link finder
- **Process:**
  1. Install extension
  2. Visit resource pages
  3. Click extension to highlight broken links
  4. Note broken links in target niche

#### 4. **Screaming Frog SEO Spider**
- **Purpose:** Crawl entire sites for broken links
- **Process:**
  1. Crawl competitor sites
  2. Export broken links report
  3. Filter for marketing/AI/lead gen niches
  4. Prioritize high-authority pages

#### 5. **Hunter.io Email Finder**
- **Purpose:** Find contact info for outreach
- **Process:**
  1. Identify target domain
  2. Use Hunter to find editor/webmaster email
  3. Verify email addresses
  4. Send outreach emails

### C. Expired Domain Research

#### Method: Find Expired Domains in Marketing Niche
1. **Use ExpiredDomains.net or DomCop**
   - Filter by: Marketing, AI, Lead Generation niches
   - Minimum DA: 30+
   - Check backlink profile quality

2. **Analyze Backlinks to Expired Domain**
   - Use Ahrefs to see all referring domains
   - Export list of sites linking to expired domain
   - Categorize by relevance to Capture Client

3. **Create Replacement Content**
   - Match topic of expired domain's content
   - Improve quality (Skyscraper technique)
   - Pitch to referring domains as replacement
