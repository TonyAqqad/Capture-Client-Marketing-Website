## Color Psychology

### Cyan/Blue (Primary, Accent)
- Trust, professionalism, technology
- Used for: Primary CTAs, trust badges, stats

### Green
- Safety, guarantee, approval
- Used for: Risk reversal, checkmarks, verified badges

### Orange/Amber
- Urgency, warning, attention
- Used for: Capacity indicator, scarcity messaging

### Yellow
- Achievement, excellence, awards
- Used for: Star ratings, excellence badge

### Gray/White
- Clean, modern, professional
- Used for: Text, backgrounds, borders

---

## Responsive Behavior

### Desktop (> 768px)
- All components visible as designed
- Multi-column layouts
- Hover effects active
- Mobile CTA bar hidden

### Mobile (< 768px)
- Single column layouts
- Touch-optimized interactions
- Larger tap targets (44px minimum)
- Mobile CTA bar visible after scroll
- Simplified trust signals grid (2 columns)

## Animation Details

### Entry Animations
- **Fade in**: Social Proof Banner (700ms)
- **Slide up**: Optimized Lead Form steps (300ms)
- **Scale**: Trust badge hover (300ms)

### Scroll Triggers
- Mobile CTA Bar: Appears at 500px scroll
- Smooth slide-up from bottom (300ms)

### Interactive Animations
- **Pulse**: Capacity indicator dot (continuous)
- **Ping**: Capacity indicator background (continuous)
- **Expand/Collapse**: Objection handler accordion (300ms)
- **Scale**: Button hover effects (300ms)

### Loading Animations
- **Progress bar**: Form step indicator (animated)
- **Avatar stack**: Staggered display
