# Mobile Logo Visual Guide

## The New Transparent Logo

### Visual Description

```
     ╔════════════════════════╗
     ║                        ║
     ║    ┌──┐        ┌──┐    ║
     ║    │            / │    ║  ← Corner brackets
     ║                        ║    (capture frame)
     ║                        ║
     ║       ╭───────╮        ║  ← Outer ring (faint)
     ║      ╱         ╲       ║
     ║     │   ╭───╮   │      ║  ← Main focus ring
     ║     │  ╱  •  ╲  │      ║  ← "C" monogram + target dot
     ║     │ ╰───────╯ │      ║
     ║      ╲         ╱       ║
     ║       ╰───────╯        ║
     ║                        ║
     ║    │            / │    ║
     ║    └──┘        └──┘    ║  ← Corner brackets
     ║                        ║
     ╚════════════════════════╝
```

### Color Scheme

**Cyan Gradient Theme:**
- Primary: `#17B4EF` (Bright Cyan)
- Mid: `#0EA5E9` (Sky Blue)
- Accent: `#06B6D4` (Teal)

**Usage:**
- Outer ring: Faint gradient (40% opacity)
- Focus ring: Bold gradient
- Corner brackets: Bold gradient
- "C" shape: Gradient stroke
- Center dot: Solid #17B4EF

### Design Elements

1. **Outer Ring** (r=18px)
   - Subtle capture boundary
   - 40% opacity for depth
   - Gradient stroke

2. **Corner Brackets**
   - L-shaped paths at 4 corners
   - Suggest camera viewfinder
   - "Capture" visual metaphor

3. **Main Focus Ring** (r=14px)
   - Bold 2.5px stroke
   - Primary visual element
   - Creates depth with outer ring

4. **"C" Monogram**
   - Partial circle (270° arc)
   - Represents "Capture Client"
   - Subtle branding element

5. **Target Dot** (r=2.5px)
   - Solid cyan center
   - "Target acquired" metaphor
   - Focal point

### Technical Excellence

- **100% Transparent** - No background fill
- **Pure Vector** - No embedded images
- **Scalable** - Looks sharp at any size
- **Optimized** - Only 1.9KB (99.4% smaller)
- **Accessible** - High contrast on dark/light
